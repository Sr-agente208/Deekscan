DETECÇÃO DE DEEPFAKES

Estrutura Técnica — Trabalho de Conclusão de Curso

1 Métodos de Detecção

1.1 Análise de Frames (CNN)

Extração e análise de frames individuais processados por redes neurais convolucionais.

  • Olhos — padrões irregulares de piscar
  • Boca — dessincronização labial
  • Textura da pele — artefatos de síntese por IA
  • EfficientNet / Xception — transfer learning
💡 Por que modelos pré-treinados?

O diferencial está na pipeline completa: integração, análise, métricas e discussão ética.

1.2 Análise Temporal

Consistência entre frames consecutivos. Deepfakes apresentam inconsistências sutis de iluminação e geometria facial.

2 Bases de Dados
  • FaceForensics++ — padrão da área
  • DFDC — Facebook/Meta
  • Celeb-DF — alta qualidade visual
  • WildDeepfake — alta variabilidade
⚠️ Balanceamento

Use oversampling, undersampling ou class weights para evitar viés.

Como obter suas chaves API

Siga o passo a passo para cada provedor e cole as chaves no Analisador

🤖
Anthropic Claude
claude-opus-4 · claude-sonnet-4
Pago por uso
🚀 Abrir Console Anthropic ↗
1
Criar conta
Acesse platform.anthropic.com e clique em Sign Up. Pode usar e-mail ou conta Google.
2
Adicionar crédito
Vá em Settings → Billing e adicione um cartão. O mínimo é $5. Há créditos gratuitos iniciais.
3
Gerar a chave
No menu lateral, clique em API Keys → + Create Key. Dê um nome (ex: "DeepScan") e confirme.
4
Copiar e salvar
⚠️ A chave aparece uma única vez. Copie e guarde em local seguro. Começa com sk-ant-api03-...
5
Colar no DeepScan
Vá em Analisar → Configure as IAs → campo Claude e cole sua chave.
💡
Custo estimado: cada análise custa aproximadamente $0.01–0.03 dependendo do modelo.
🧠
OpenAI GPT-4o
gpt-4o · visão computacional
Pago por uso
🚀 Abrir Plataforma OpenAI ↗
1
Criar conta
Acesse platform.openai.com e registre-se. Pode usar e-mail, Google ou Microsoft.
2
Adicionar crédito
Vá em Settings → Billing → Add payment method. Adicione mínimo $5. Sem crédito a chave retorna erro de quota.
3
Criar a chave
No menu lateral, clique em API Keys → + Create new secret key. Dê um nome e defina as permissões como All.
4
Copiar e salvar
⚠️ A chave aparece uma única vez. Começa com sk-proj-.... Guarde imediatamente.
5
Colar no DeepScan
Vá em Analisar → Configure as IAs → campo GPT-4o e cole sua chave.
⚠️
Erro de quota? Isso significa que não há crédito na conta. Adicione pelo menos $5 em Settings → Billing para ativar.
🔍
IsItAI Detector
isitai-detect-v1 · especializado
Tem plano grátis
🚀 Abrir IsItAI.com ↗
1
Criar conta
Acesse isitai.com e clique em Sign Up. É gratuito para começar.
2
Acessar o painel
Após o login, vá no menu do usuário e acesse Dashboard → API ou Developer.
3
Gerar a chave
Clique em Generate API Key. A chave começa com isitai-....
4
Colar no DeepScan
Vá em Analisar → Configure as IAs → campo IsItAI e cole sua chave.
✅
Detector especializado: O IsItAI é treinado especificamente para detectar imagens geradas por IA — excelente para complementar o Claude e o GPT.
